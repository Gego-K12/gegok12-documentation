# Events & calendar Management

1.	Click calendar from the left pane of the admin dashboard 


![Screenshot](./images/event/e1.png)

2.	Click Create Events on the right top corner 

![Screenshot](./images/event/e2.png)

<span style="color:red">Note: Event can be classroom event, School event or Alumni event</span>

3.	Click Classroom Event from Create event tab

![Screenshot](./images/event/e3.png)

4.	Fill all the details and click submit 

![Screenshot](./images/event/e4.png) 

5.	Classroom event will be listed in the calendar as shown below

![Screenshot](./images/event/e5.png) 

<span style="color:red">Note: The same procedure will be applied for School Event and alumni event . All the events will be listed in the digital school calendar</span>
