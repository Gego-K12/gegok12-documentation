# Telephone Directory Management    
 

![Screenshot](./images/telephone-directory/t1.png)

1.	Click Telephone Directory on the left pane 

![Screenshot](./images/telephone-directory/t2.png)

2.	Click Add phone number tab from the right top corner 

![Screenshot](./images/telephone-directory/t3.png)

3.	A new from will open, record all the details in the form and click submit. 

<span style="color:red">Note: You can also Reset the same if there are any deletions</span> 

![Screenshot](./images/telephone-directory/t4.png) 

4.  New phone number with all the details will be listed down in the digital telephone directory. The same can be used anytime and from anywhere. 

<span style="color:red">Note: Editing and deleting is also possible from the Actions tab</span>
