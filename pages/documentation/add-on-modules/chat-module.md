# Chat Module (Pro)

The Chat Module (Pro) provides secure, real-time messaging between teachers, students, parents, and administrators within the school platform. It fosters effective communication, reduces delays, and ensures all stakeholders stay informed and connected.

---

## Installation

To install the Chat  Module, run:

```bash
php artisan module:install-chat
```

---

## GitHub Authentication

Since this module is stored in a **private GitHub repository**, you'll need a **GitHub username and personal access token** after purchasing the module.

When prompted:

```bash
Username for 'https://github.com': your-github-username
Password for 'https://github.com': your-github-token
```

>  **Note**: The token should have at least `read:packages` and `repo` scopes.

## Troubleshooting

- **403 error on GitHub?**  
  Double-check your token scope or expiration.

- **Module not found?**  
  Ensure your license is active and GitHub access has been granted.
