# Transport Module (Pro)

The Transport Module (Pro) enables schools to efficiently manage student transportation, including route planning, vehicle tracking, and stop-wise assignments. It provides real-time visibility for administrators and ensures better coordination and safety for daily school commutes.

---

## Installation

To install the Transport Module, run:

```bash
php artisan module:install-transport
```

---

## GitHub Authentication

Since this module is stored in a **private GitHub repository**, you'll need a **GitHub username and personal access token** after purchasing the module.

When prompted:

```bash
Username for 'https://github.com': your-github-username
Password for 'https://github.com': your-github-token
```

>  **Note**: The token should have at least `read:packages` and `repo` scopes.

##  What’s Included

- Vehicle and driver management
- Route and stop creation with pickup/drop timing
- Student-to-route/stop assignment
- Transport attendance 
- Printable route sheets and ID cards
- Notifications to parent pick up and drop

## Troubleshooting

- **403 error on GitHub?**  
  Double-check your token scope or expiration.

- **Module not found?**  
  Ensure your license is active and GitHub access has been granted.
