# List of Modules

GegoK12 is designed with a flexible modular architecture that allows schools to start with essential features and expand as needed. Our modules are divided into two categories: **Free Core Modules** and **Paid Add-on Modules**.

## Free Modules (Core Modules)

When you pull the code and set up GegoK12, you'll have access to all **Core Modules** by default. These modules provide comprehensive functionality that is **enough for a school to run** its day-to-day operations efficiently.

### Core modules included:

| Module | Description |
|--------|-------------|
| **[Settings](core-modules/settings)** | Configure system-wide settings, academic years, and school information |
| **[Holiday Management](core-modules/holiday-list)** | Manage holidays, school calendar, and academic schedules |
| **[Leave Master](core-modules/leave-master)** | Handle staff and student leave requests and approvals |
| **[Staff Management](core-modules/staff-management)** | Manage staff profiles, roles, departments, and payroll |
| **[Class Management](core-modules/class-section-management)** | Create and manage classes, sections, and classroom assignments |
| **[Subject Management](core-modules/subject-management)** | Define subjects, assign teachers, and manage curriculum |
| **[Student Management](core-modules/student-management)** | Complete student information system with profiles and records |
| **[Attendance Management](core-modules/attendance-management)** | Track daily attendance for students and staff |
| **[Homework Management](core-modules/homework-management)** | Assign, submit, and evaluate homework assignments |
| **[Communication Management](core-modules/communication-management)** | Send notifications, emails, and SMS to parents and staff |
| **[Lesson Management](core-modules/lesson-management)** | Plan and organize lesson plans and teaching schedules |
| **[ID Card & Bus Pass](core-modules/id-card)** | Generate and print student ID cards and bus passes |
| **[Dashboard & Reports](core-modules/dashboard-management)** | Comprehensive analytics and reporting dashboard |
| **[Library Management](core-modules/library-management)** | Manage library books, issue/return tracking, and cataloging |

| **[Event & Calendar](core-modules/event-management)** | Schedule and manage school events and activities |

## Paid Modules (Add-on Modules)

Add-on Modules are **advanced features** designed for schools that need specialized functionality beyond the core system. These modules enhance GegoK12 with powerful tools for assessments, management, and communication.

### Available Add-on Modules:

| Module | Description | Key Features |
|--------|-------------|--------------|
| **[Exam Module](add-on-modules/exam-module)** | Comprehensive examination system | Create exams, manage question banks, automated grading, result processing, grade cards, mark sheets |
| **[Fee Module](add-on-modules/fee-module)** | Advanced fee management system | Fee structure setup, online payments, installments, reminders, receipts, financial reports |
| **[Alumni Module](add-on-modules/alumni-module)** | Alumni relationship management | Alumni database, event management, networking, career tracking, fundraising campaigns |
| **[Video Room Module](add-on-modules/video-room-module)** | Integrated video conferencing | Live classes, online meetings, screen sharing, recording, breakout rooms, attendance tracking |
| **[Transport Module](add-on-modules/transport-module)** | School transportation management | Route planning, vehicle tracking, driver management, GPS integration, pickup/drop notifications |
| **[Quiz Module](add-on-modules/quiz-module)** | Interactive quiz and assessment tool | Create quizzes, multiple question types, timed tests, instant feedback, performance analytics |
| **[Inventory Module](add-on-modules/inventory-module)** | Stock and asset management | Track school assets, inventory levels, purchase orders, vendor management, maintenance schedules |
| **[Chat Module](add-on-modules/chat-module)** | Real-time messaging system | Group chats, private messaging, file sharing, notifications, teacher-parent-student communication |
| **[Certificate Module](add-on-modules/certificate-module)** | Certificate generation system | Custom templates, bulk generation, digital certificates, verification system, automatic distribution |

### Benefits of Add-on Modules

- **Specialized Functionality** - Advanced features for specific needs
- **Scalability** - Add modules as your school grows
- **Cost-Effective** - Pay only for what you need
- **Seamless Integration** - Works perfectly with Core Modules
- **Regular Updates** - Continuous improvements and new features
- **Dedicated Support** - Priority support for paid modules
