# Inventory   Module (Pro)

The Inventory Module (Pro) allows schools to efficiently track and manage their physical assets such as books, uniforms, lab equipment, electronics, and stationery. It helps ensure accountability, minimize losses, and maintain organized stock records across departments.

---

## Installation

To install the Inventory  Module, run:

```bash
php artisan module:install-inventory
```

---

## GitHub Authentication

Since this module is stored in a **private GitHub repository**, you'll need a **GitHub username and personal access token** after purchasing the module.

When prompted:

```bash
Username for 'https://github.com': your-github-username
Password for 'https://github.com': your-github-token
```

>  **Note**: The token should have at least `read:packages` and `repo` scopes.

## Troubleshooting

- **403 error on GitHub?**  
  Double-check your token scope or expiration.

- **Module not found?**  
  Ensure your license is active and GitHub access has been granted.
