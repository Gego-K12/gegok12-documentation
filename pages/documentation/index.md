# Introduction

## What is gegok12.com?

## Key Features

- Teacher & Student Portals
- Calendar Management
- Announcements and Notifications
- Lesson Planning Tools
- Academic Reports 
- Role-based Access Control
- File & Resource Sharing
- Parent-Teacher Communication
- Homework and assignment

---

## Who is it for?

- **Superadmin**  
   Manage multiple schools or branches, control global settings and oversee tenant-level activity across the entire system.

- **Admin**  
  Manage school settings, configure roles and permissions, oversee system-wide operations

- **Teachers**  
  Create classes, manage students, assign homework and track performance with ease.
- **Students**  
  Access class materials, complete assignments, view schedules, and monitor academic progress.

- **Parents**  
  Stay informed about their child's attendance, grades, upcoming events, and directly connect with teachers by using app

- **Librarians**  
   Manage book inventory, issue/return logs, overdue tracking, and student library access.

- **Receptionists**  
   Handle inquiries, manage front desk operations, student records, and parent walk-ins.

- **Accountants**  
   Monitor and record fee transactions, generate financial reports, manage dues, and oversee fee structures.
